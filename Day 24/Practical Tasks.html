1.Define an error in JavaScript

An error in JavaScript is an unexpected condition or mistake in the code that stops normal execution and may cause the program to crash or behave incorrectly.

2.List types of JavaScript errors

  1. Syntax Error
  2. Reference Error
  3. Type Error
  4. Range Error
  5. URI Error
  6. Eval Error

3.Difference between runtime and logical error

Runtime Error: An error that occurs while the program is running and stops execution.
Logical Error: An error in program logic that produces incorrect output without stopping execution.

4.Explain try/catch/finally

try: Used to write the code that might cause an error.
catch :Used to handle the error if something goes wrong in try.
finally : Used to write code that always runs, whether an error occurs or not.

5.What is the use of throw?

throw is used to generate a custom error intentionally when a specific condition is met.
