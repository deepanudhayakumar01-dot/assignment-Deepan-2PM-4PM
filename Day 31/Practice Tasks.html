1.Define React in your own words

React lets you create reusable UI components and update only the parts of the page that change, instead of reloading the entire page.


2.List two advantages of React

1.Reusable Components
React allows you to build small, reusable components, which makes code easier to maintain, test, and scale.
2.Fast Performance
React uses a Virtual DOM to update only the parts of the UI that change, resulting in faster and more efficient rendering.


3.Identify entry file in React project

The entry file in a React project is index.js (or main.jsx in Vite), where the app is rendered into the DOM.
